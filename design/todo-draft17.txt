Outline of changes needed to move from IETF Java LDAP API draft
version 13 to version 17.

See: http://www.ietf.org/internet-drafts/draft-ietf-ldapext-ldap-java-api-17.txt

Note: The number followed by ":" indicates the draft revision
where the change was made.

The [ ] can contain the following
  - (blank) no one working on feature
r - (any letter except X) - initial of person working of feature
X - (capitol X) - complete
